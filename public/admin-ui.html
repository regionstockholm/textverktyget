Automatisk profilväxling (quality/balanced/stress)

Växlar profil efter belastning. Auto följer mätvärden, Manual låser vald profil. Dry run loggar förslag utan att tillämpa dem.

1) Driftläge: Auto enabled slår på av/på hela funktionen. Mode styr om växling sker automatiskt eller manuellt. Manual profile används endast när Mode = manual. Dry run testar logik utan att ändra profil.

2) Utvärderingsintervall: Styr hur ofta systemet räknar på belastning och hur länge ett mätfönster är aktivt.

3) Växlingsregler: Definierar när profilbyte får ske baserat på antal mätpunkter och antal på varandra följande signaler.
